Linora Cipher

A modular difference-chain cipher by modify

Overview

The Linora Cipher is a symmetric encryption algorithm created by modify in 2025. It transforms characters using modular arithmetic and chained differences, making each output character reliant on both the input and the previous result.

Algorithm Description

  1. Convert all characters to numbers: a = 0, b = 1, ..., z = 25.
  2. First character:
    • Add the first key element to it (modulo 26).
  3. Each subsequent character:
    • Compute the difference from the previous character (mod 26).
    • Multiply the result by the corresponding key element (cycled).
    • Take the result modulo 26.
  4. To decode:
    • Subtract the first key element from the first character (mod 26).
    • For the rest, multiply by the modular inverse of each key element and reverse the chaining.
